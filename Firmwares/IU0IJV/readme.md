2. visit: https://egzumer.github.io/uvtools/<br>
3. click browse and select the downloaded firmware<br>
4. click "Flash firmware"<br>
5. connect and put radio in flashmode (hold PTT and turn-on radio)<br>
6. flash<br><br><br>
